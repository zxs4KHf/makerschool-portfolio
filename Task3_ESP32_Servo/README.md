# 基于 ESP32 的 WiFi 智能舵机控制系统

## 项目说明
结合课堂发放的元器件，利用 ESP32 的 **WiFi 模块** 与 **PWM (模拟输出) 控制舵机** 两个基础功能，实现了一个通过网页局域网控制的智能舵机。

## 硬件与接线
1. ESP32 开发板
2. SG90 舵机

- 舵机红色引脚连接 ESP32 5V (或 3.3V 视情况而定)
- 舵机棕色引脚连接 ESP32 GND
- 舵机橙色信号引脚连接 ESP32 GPIO 13

## 软件实现说明
利用 `WiFi.h` 库连接局域网，利用 `WebServer.h` 建立一个运行在 80 端口的轻量级 Web 服务器。利用 `ESP32Servo.h` 库输出不同占空比的 PWM 信号，从而控制舵机转动至指定角度 (0°, 90°, 180°)。

> 注：因实习冲突，本项目仅提供详细的底层源码设计与硬件连线架构规划，未能在学校实验室完成物理硬件组装验证。
