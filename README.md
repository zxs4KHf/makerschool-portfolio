## 📂 仓库结构说明

本项目采用严格的工程规范进行目录管理，各个任务与项目的源码和设计文件分布如下：

- **`index.html` & `style.css`**：项目总览的主页源代码（Weblog）。
- **`Task2_3D_Design/`**：交大130周年纪念文创 3D 建模及实物图，包含 `.stl` 源文件。
- **`Task3_ESP32_Servo/`**：基于 ESP32 的 WiFi 智能舵机控制系统 C++ 源码。
- **`Task4_LED_Badge/`**：交大校庆 LED 电子徽章 Altium Designer PCB 布线图及 3D 渲染图。
- **`Task5_TP4056/`**：TP4056 锂电池充电管理模块电路分析及管脚定义表。
- **`Task6_Mortise_Tenon/`**：传统榫卯装置 3D 结构设计及渲染图。
- **`MidTerm_SmartClock/`**：中期创客项目 —— 基于 ESP32S2 的智能桌面时钟信息终端（包含 3D 外壳设计与联网代码）。
- **`Final_UR5_Grasping/`**：期末黑客松项目 —— UR5与六自由度灵巧手的自适应抓取系统核心控制管线代码 (`grasping_pipeline.py`) 及系统架构图。

> 每个具体的子项目目录中，均包含独立的 `README.md`，对核心技术和代码逻辑进行了深度说明。
