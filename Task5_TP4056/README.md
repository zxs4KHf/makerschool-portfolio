# TP4056 锂电池充电管理模块分析

## 项目说明
基于带有散热片的 TP4056 (C16581) 的理论复刻。完成了对 TP4056 管脚功能的深度分析，电阻选型计算，以及 Altium Designer 平台下的原理图与 PCB 板图布局设计。

> 注：因实习冲突未能返校，本项目提供了详尽的 EDA 理论设计图纸、关键管脚与元件参数选型，未进行实物的打样焊接。

## 1. 核心元件选型
1. **充电电流配置电阻 (R_PROG):** 充电电流由 PROG 管脚的电阻决定，公式为 $I_{CHG} = \frac{1200}{R_{PROG}}$。为了设定 1A 的满载充电电流，选用 $R_{PROG} = 1.2k\Omega$ 的精密电阻（精度1%）。
2. **旁路滤波电容:** VCC 输入端需要 10µF 的旁路电容，用于吸收输入电源的纹波和高频噪声，保证芯片供电稳定；BAT 端并联 10µF 陶瓷电容，用于在电池未连接或电池内阻较大时，平移输出电压纹波并防止震荡。

## 2. 管脚功能表

| 管脚 | 名称 | 功能描述 |
| --- | --- | --- |
| 1 | TEMP | 电池温度检测输入端。若接低电平或高电平则暂停充电，接GND时功能禁用。 |
| 2 | PROG | 恒流充电电流设置端与充电电流监测。通过外接电阻至地来设定恒流充电电流。 |
| 3 | GND | 电源地。全板铺铜连接的网络。 |
| 4 | VCC | 正输入供电电压。当VCC降至BAT以下，TP4056进入低功耗睡眠模式。 |
| 5 | BAT | 电池连接端。为电池提供充电电流和限制电压。 |
| 6 | STDBY | 电池充电完成指示端。充电完成时内部开关拉低。 |
| 7 | CHRG | 充电状态指示端。充电中被拉低，否则呈高阻态。 |
| 8 | CE | 芯片使能输入端。高电平使能，低电平使芯片休眠。 |

## 3. PCB 布局说明
在 Altium Designer 的设计中，vcc 线进行了适当的实线填充以适应大电流，全板铺铜连接 GND 网络以增强散热和抗干扰能力。边框层设计为圆角矩形，并通过了完整的 DRC 检查。
