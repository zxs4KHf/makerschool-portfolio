# 基于 ESP32S2 的智能桌面时钟信息终端

## 项目背景 (中期创客项目)
融合3D设计与打印、电子焊接、嵌入式计算等技术，设计了一个实物作品的**全套软硬件方案**。受限于本人外出实习的时间冲突，本项目未能在实验室完成最终的 3D 外壳打印与电路板焊接，但在此提交了严谨的**硬件架构设计**、**3D 打印外壳渲染图**以及核心的**软件系统源码**。

## 硬件设计思路
1. **主控**：ESP32-S2 开发板（符合课程允许的平台要求）
2. **显示**：0.96寸 OLED i2c 接口屏幕
3. **传感器**：DS18B20 温度传感器（通过单总线协议获取环境温度）
4. **结构件**：采用 3D 建模设计的倒角矩形外壳，内含主板固定柱与走线槽。

## 软件功能描述
程序采用 Arduino 框架编写。系统上电后自动通过 WiFi 连接互联网，利用 NTP 协议获取实时时间。同时读取 DS18B20 传感器的环境温度，将时间、日期、温度信息通过 U8g2 库渲染并显示在 OLED 屏幕上。
